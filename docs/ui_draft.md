# 页面草图

UI 参考 ANI-RSS 的“订阅、下载、资料、洗版、设置”信息组织方式，但不复制源码、素材或默认来源。

| 页面 | 主要内容 | 必备操作 |
| --- | --- | --- |
| 任务看板 | 今日新增、待确认、下载中、整理中、失败任务 | 查看详情、重试、人工确认 |
| 来源管理 | 授权来源、抓取间隔、指纹规则、标题清洗 | 新增、停用、测试、删除 |
| 匹配中心 | 候选剧名、年份、季集、匹配分 | 选择资料、保存规则 |
| 下载队列 | 下载器状态、速度、进度、文件清单 | 暂停、继续、重新分析 |
| 文件分析 | 正片、字幕、样片、说明文件、排除文件 | 调整类型、调整下载优先级 |
| 命名预览 | 原路径、新路径、冲突、非法字符 | 执行入库、返回修改 |
| 入库记录 | 执行时间、旧路径、新路径、结果 | 回滚、查看日志 |
| 系统设置 | 下载器、TMDB、媒体库、阈值、通知、备份 | 测试连接、保存、恢复 |
| 运行日志 | 任务编号、阶段、动作、结果、失败原因 | 筛选、导出、清理 |
