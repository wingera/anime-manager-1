# 术语表

| 代码词 | 中文含义 | 使用位置 |
| --- | --- | --- |
| source | 来源，用户有权使用的资源入口 | 来源管理、数据库、接口 |
| resource | 资源，一条待处理的资源记录 | 任务流、数据库、接口 |
| resource_hash | 资源指纹，40 位十六进制字符串 | 资源识别、去重 |
| magnet_uri | 磁力入口，下载器能识别的下载入口 | 下载任务 |
| metadata | 影视资料，剧名、年份、季集、简介等信息 | TMDB 对接、匹配中心 |
| match_score | 匹配分，判断文件对应季集的可信程度 | 自动处理、人工确认 |
| download_task | 下载任务，下载器中的任务映射 | 下载队列 |
| media_file | 媒体文件，下载任务内的具体文件 | 文件分析、命名预览 |
| import_record | 入库记录，移动、复制或硬链接的执行记录 | 入库记录、回滚 |
| season_number | 季号 | 季集判断、命名 |
| episode_number | 集号 | 季集判断、命名 |
